Introduction

cowsay.rest provides a contemporary, RESTful solution to having ASCII art cows to say things.

API

Path Verb Description
/api/cows GET

List the available cows
Results:
/api/say/:cow/:eyes/:tongue GET

Say "Moo" for the optional cow, eyes, and tongue
/api/say/:cow/:eyes/:tongue POST

Say posted content for the optional cow, eyes, and tongue. Valid payloads are a string of content for content-type text/plain or json formatted as follows for application/json: { "say": "Moo" }
